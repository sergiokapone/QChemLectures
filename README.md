# Квантова хімія та квантово-механічні обчислення


Дисципляна "Квантова хімія та квантово-механічні обчислення" використовує положення квантової механіки для інтерпретації явищ, 
що протікають в атомах, молекулах та твердих тілах з використанням сучасних програм обчислення та візуалізації.

Стрімкий розвиток сучасних технологій швидко змінює структуру промислового виробництва і 
ставить перед людством задачу підготовки сучасних фахівців в галузі прикладної науки, 
які здатні адаптуватися і орієнтуватися в швидких змінах наукоємних високих технологіях. 
Вивчення дисципліни має на меті донесення знань про принципи і методи квантової хімії до фахівця в галузі прикладної фізики, 
що дасть йому змогу застосовувати їх при створенні речовин і матеріалів з наперед заданими властивостями та у інших наукоємних технологіях.
